The Variable “INFO” contains multiple fields (AKA Extensible meta-data), each separated by a semi-colon.

The Variable “FORMAT” again contains multiple fields, each separated by a semi-colon and it directly refers to the format of the subsequent fields: “NA12878”, “NA12891”, “NA12892”. Each of these is a different sample with the values of the variables described by FORMAT
